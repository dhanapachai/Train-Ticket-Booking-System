# RailBook — Train Ticket Booking System

A full-stack train booking web application built with React + Spring Boot + MySQL. Supports user registration, train search, multi-class seat booking, automatic waiting list management, and a complete admin panel.
Tech Stack:

Frontend: React 18, React Router v6, Axios, Vite

Backend: Java 17, Spring Boot 3.2, Spring Security 6, JWT

Database: MySQL 8, Spring Data JPA, Hibernate

# Features:
•	JWT authentication with role-based access (USER / ADMIN)

•	Train search with source, destination, date filters

•	Real-time seat availability across UPPER, LOWER, MIDDLE classes

•	Automated waiting list with auto-promotion on cancellation

•	Admin dashboard: Add/Edit/Delete trains, view booking stats
